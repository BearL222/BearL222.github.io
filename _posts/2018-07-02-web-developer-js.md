## Logical Operations

AND, OR, NOT e.g.  

|	Operator	|	Name	|	Example	|	Result	|
|---|---|---|---|
|&&|AND|x < 10 && x !== 5|false|
|\|\||OR|y < 9 ``||`` x === 5|true|
|!|NOT|!(x === y)|true|

1. AND: &&
2. OR: OR
3. NOT: !
4. Truthy and Falsy values:  
Values that aren’t actually true of false, are still inherently “truthy” or “falsy” when evaluated in a boolean context  
e.g. !”hello” = False

Falsy values:

- false
- 0
- ""
- null
- undefined
- NaN

Everything else is truthy
